Module 1: Introduction to Generative AI

Generative AI refers to algorithms capable of generating new content like text, images, music, etc.

Examples include GPT, DALL·E, and StyleGAN.

This module covers the basics of generative AI, its history, and applications.

Module 2: Types of Generative Models

  • Variational Autoencoders (VAE)
  • Generative Adversarial Networks (GANs)
  • Autoregressive Models (e.g., GPT)

Each type has strengths, weaknesses, and unique use cases.

Module 3: Understanding GANs

GANs consist of a generator and discriminator in competition.

The generator tries to create realistic outputs, the discriminator tries to detect fakes.

This adversarial process improves both components.

Module 4: Applications of Generative AI

  • Text generation (e.g., ChatGPT)
  • Image synthesis (e.g., DALL·E)
  • Code generation (e.g., GitHub Copilot)
